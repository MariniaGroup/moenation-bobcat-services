# Moenation Bobcat Services

Contractor-focused landing page for Moenation Bobcat Services.

## Planned features

- Responsive React/Vite landing page
- Rust, black, cream, and white branding
- Contractor-focused service messaging
- Quote-request form
- AWS Amplify deployment
- Analytics integration
- Future connection to the contractor lead-intelligence project

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
```
